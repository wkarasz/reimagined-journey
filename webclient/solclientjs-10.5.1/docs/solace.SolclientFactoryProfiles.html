namespace solace.SolclientFactoryProfiles

The collection of predefined factory profiles available for application use.

See each member for a description of its configuration.

Namespaces Top

version7

The Version 7 profile for SolclientJS. This profile sets factory and session default values that provide SolclientJS 7.x behaviour, and interoperability with SolclientJS 7.x clients.
version10

The Version 10 profile for SolclientJS. This profile sets factory and session default values that provide the full capability of the API as of version 10.0, including interoperability with other Solace Messaging products.
version10_5

The Version 10.5 profile for SolclientJS. This profile sets factory and session default values that provide the full capability of the API as of version 10.5, including interoperability with other Solace Messaging products.
