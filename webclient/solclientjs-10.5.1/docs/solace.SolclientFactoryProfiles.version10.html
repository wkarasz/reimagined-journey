namespace solace.SolclientFactoryProfiles.version10

The Version 10 profile for SolclientJS. This profile sets factory and session default values that provide the full capability of the API as of version 10.0, including interoperability with other Solace Messaging products.

static public readonly Boolean byteArrayAsString

Returns true: getValue() of SDTFields of type BYTEARRAY return a string, and so does getBinaryAttachment().

static public readonly Boolean cometEnabled

Returns false: Comet HTTP transports are disabled for the API. The API will not select or fall back to HTTP Comet transport protocols, which are mutually exclusive of Guaranteed Messaging features.

static public readonly Boolean guaranteedMessagingEnabled

Returns true: Guaranteed Messaging is enabled for the API, with the result that solace.MessagePublisherProperties#enabled is true by default on platforms that support Guaranteed Messaging publishing.

Where the solace.TransportType.WS_BINARY transport is unavailable, solace.MessagePublisherProperties#enabled is false by default.

static public readonly Boolean guaranteedMessagingEnabled

Returns true: Guaranteed Messaging is enabled for the API, with the result that solace.MessagePublisherProperties#enabled is true by default on platforms that support Guaranteed Messaging publishing.

Where the solace.TransportType.WS_BINARY transport is unavailable, solace.MessagePublisherProperties#enabled is false by default.

static public readonly Boolean topicUtf8Encode

Returns true: the API will encode topics for interoperability with other Solace Messaging products. SolclientJS 7.x clients may fail to correctly decode when topic names that include multi-byte UTF-8 code points.
